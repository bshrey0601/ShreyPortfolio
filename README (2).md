## Tech Stack

- **Frontend:** React 18, Tailwind CSS, Babel (zero-build, CDN-based)
- **Typography:** Fraunces (display), Hanken Grotesk (body), JetBrains Mono (code)
- **Visualizations:** Custom SVG charts (bar, donut, sparkline, football-field, bump) — all built from scratch in React
- **Deployment:** GitHub Pages (static single-file HTML)

---

## Features

- Animated hero section with profile photo and rotating tagline
- Scroll-triggered reveal animations and count-up stats
- Interactive project cards with expandable details and inline data visualizations
- Responsive design — works on desktop, tablet, and mobile
- Smooth scroll navigation with active section tracking
- Premium boot loader and scroll progress indicator

## Run Locally

No build step required. Just open the file in a browser:

```bash
# Clone the repo
git clone https://github.com/bshrey0601/bshrey0601.github.io.git

# Open in browser
open index.html
```

---

## Deploy to GitHub Pages

1. Create a repository named `bshrey0601.github.io`
2. Push `index.html` to the `main` branch
3. Site goes live at `https://bshrey0601.github.io`
